### About This Course

This course uses **Paulo Freire's pedagogy of the oppressed** — education is not a banking transaction but a dialogue between equals. Each module begins with a *generative theme* drawn from real working-class experience in Aotearoa, followed by critical reflection and practical action steps (praxis).

Bilingual te reo Maori / English throughout. Black and white design. Free forever.

### Pedagogy

This course is built on Paulo Freire's *Pedagogy of the Oppressed* (1968). The "banking model" of education — where the teacher deposits knowledge into passive students — is rejected. Instead, each module:

1. Opens with a **generative theme** rooted in lived experience
2. Provides **critical content** (theory, law, examples)
3. Closes with a **praxis box** — reflect, discuss, act
